7 DAYS OF CODE

O 7 Days of Code consiste em um projeto executado ao longo de 7 dias após a finalização de todos os cursos da formação, com o objetivo de fixar o conteúdo e tentar alguns códigos novos. O foco do 7 Days of Code é javaScript, com os códigos inseridos nessa página a fim de documentar a jornada.
